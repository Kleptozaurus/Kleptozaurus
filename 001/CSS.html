CSS (52.97%)

Sukūrimo data: 1995

CSS (Cascading Style Sheets) yra kalba, skirta aprašyti tinklalapių išdėstymą ir stilių. Ji leidžia nustatyti šriftus, spalvas, pozicijas ir kitus dizaino elementus. CSS naudojamas kartu su HTML ir JavaScript, o jame taikomos taisyklės nurodo, kaip turinys turi būti rodomas naršyklėje.
