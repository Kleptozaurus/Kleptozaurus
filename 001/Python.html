Python (49.28)

Sukūrimo data:

Python yra aukšto lygio interpretuojama programavimo kalba, sukurta Guido van Rossum. Ji pasižymi paprastumu, skaitomumu ir dideliu populiarumu dėl plačios pritaikomumo srities, nuo internetinių tinklalapių iki dirbtinio intelekto. Python palaiko įvairias paradigmas, įskaitant objektinį, funkcinį ir procedūrinį programavimą, ir turi gausią trečiųjų šalių bibliotekų ekosistemą.
