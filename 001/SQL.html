SQL (48.66%)

Sukūrimo data: 1995

SQL (Structured Query Language) yra duomenų bazių valdymo kalba, naudojama duomenų išgavimui, manipuliavimui ir valdymui. Ji leidžia kurti, keisti ir valdyti duomenų bazės schemą, vykdyti užklausas ir analizuoti duomenis. SQL yra standartinė ir plačiai naudojama kalba duomenų bazių sistemose.
