Sukūrimo data: 1995

JavaScript yra universalioji programavimo kalba, plačiai naudojama tiek naršyklėse, tiek serveriuose, kurią sukūrė Netscape 1995 metais. Tai objektinė ir įvykių valdoma kalba,skirta interneto puslapiams dinamiškumą suteikti. Ji leidžia kurti interaktyvius svetaines, valdyti naršyklės lango turinį, atlikti asinchronines užduotis ir t. t. JavaScript yra įterptinė kalba, kuri integruojama į HTML ir gali bendrauti su kitomis interneto technologijomis.
